# Wynn SDK (JS)

Wynn SDK is a library that allows you to interact with the Wynncraft API.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install Wynn SDK.

```bash
npm i wynn-sdk-js
```

## Usage

```python
import Wynn from 'wynn-sdk-js';

const wynn = new Wynn();
```
